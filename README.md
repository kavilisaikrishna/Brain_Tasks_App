# Brain-Tasks-App

## Project Overview
Brain-Tasks-App is a React web application for managing brain tasks.  
It is containerized with Docker and deployed using Kubernetes (Minikube or AWS EKS).

## Project Files
- `dist/` - Built frontend files  
- `Dockerfile` - Docker image definition  
- `deployment.yaml` - Kubernetes Deployment  
- `service.yaml` - Kubernetes Service  
- `buildspec.yml` - AWS CodeBuild instructions
